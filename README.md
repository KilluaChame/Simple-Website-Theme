# Simple Website

This is a simple website project that demonstrates the use of light and dark themes using HTML, CSS, and JavaScript.

## Features

- Light and dark theme toggle.
- Responsive design.

## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.

## Technologies Used

- HTML
- CSS
- JavaScript
